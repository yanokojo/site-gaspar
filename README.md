# Site Gaspar - Website Template

Professional business website template in Portuguese for service-based businesses.

## Features

- **Responsive Design** - Adapts to all screen sizes
- **Hero Section** - Eye-catching introduction with WhatsApp CTA
- **Service Description** - Dedicated section for service details
- **Gallery** - Grid-based portfolio/gallery showcase
- **Testimonials** - Customer testimonial section
- **WhatsApp Integration** - Direct contact button with pre-filled message

## Quick Start

1. Open `index.html` in your browser or use VS Code Live Server
2. Replace placeholder text with your business information:
   - Company name in header
   - Service description in hero section
   - WhatsApp phone number (replace `55DDDSEUNUMERO`)
   - Service details
3. Replace placeholder images with your portfolio images

## Customization

### Update WhatsApp Link
In the hero section, modify the phone number:
```html
<a href="https://wa.me/YOUR_COUNTRY_CODE+YOUR_NUMBER?text=MESSAGE">
```

Example for Portugal: `https://wa.me/351912345678`

### Add More Testimonials
Copy and paste the blockquote block in the testimonials section:
```html
<blockquote>
    "Your testimonial text" <br>
    <strong>- Client Name ⭐⭐⭐⭐⭐</strong>
</blockquote>
```

### Replace Gallery Images
Replace placeholder URLs with your actual image URLs or local image paths.
